# Trybik and Susi's Macro

A simple Minecraft macro that automatically:
- Holds **W**
- Clicks the **left mouse button**

Mainly useful for mining (AFK).



---

## How to Use
1. Set your pickaxe slot
2. Position yourself at the mining spot
3. Press **F5** to enable the macro
4. Press **F5** again to disable

---

## Applications
Automatic tunnel mining
Examples:
- DonutSMP
- 2b2t.org
- Hypixel.net

---

## Safety
- The macro is **open-source**
- Does not connect to the internet
- Does not save files
- Does not run in the background after closing
- Only simulates keyboard and mouse

**Antivirus may show a warning** because the program simulates input
(keyboard/mouse). This is a **false positive**, common for macros.

- Bilingual support (Polish/English)
